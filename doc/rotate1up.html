piv_insort() - Insertion Sort Algorithm

SYNOPSIS

#include "piv_stdlib.h"

void piv_insort (piv_state *input, (*const comparison_f) (void*, void*),
                 const size_t type_size, const piv_table* const v_table)

DESCRIPTION

Sorts a sequential data structure using type-specfic order relation. A "less than" comparison_f function sorts input into (<) ascending order and "greater than" function sorts into descending order (>). Partial order relations (≤, ≥) produce a stable sort.

Insertion sort has in-place O(1) memory complexity and O(n^2) time complexity, but this reduces to O(n) for sequences with only a single element out of place.

piv_insort() and its macro cousin PIV_INSORT() can be applied to any sequential data structure storing any type, so long as you pass in the data structure's v_table and the data type's type_size from sizeof().

Assumes that the input container to be sorted is non-empty.

RETURN VALUE

No return value because insertion sort is in-place and never fails
